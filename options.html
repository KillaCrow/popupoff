Developer's supervision

There is list of websites where PopUpOFF is not working by default. These websites do not show popups but use same mechanic to display information to user. If you enable it there accidentally, you could hide some important information from yourself.

Disabling this "parent control" is highly not recommended. If you do it, responsibility for further using would be only on you.

List of disabled websites: YouTube Music, YouTube, LinkedIn, Twitter, Facebook, Google, Reddit, Instagram, Baidu, Amazon, Pinterest, VK
